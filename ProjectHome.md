Amazon S3 could be used to share files. However it lack—ã directory listing capability. This small utility connects to given S3 bucket and recursively generates index.html files in all sub-folders to allow easy browsing.

Generated HTML could be customized using <a href='http://www.stringtemplate.org/'>StringTemplate</a> engine.
